# 毕业照网站 - 视觉资产设计需求文档

> **用途：** 交给设计师参考，按此文档产出 SVG 图标、背景图、插画等视觉资产。

## 二、全局风格铁律

1. **所有形状必须圆润** — 禁止直角，最小圆角 12px，图标线条末端必须是圆头
2. **线条粗重** — SVG stroke-width 最小 2px，推荐 2.5-3px
3. **风格一致** — 所有图标必须是同一套"圆润可爱"的视觉语言
4. **颜色统一** — 图标主色使用 `#794f27`，点缀色使用 `#19c8b9`，背景透明
5. **无渐变**（背景图除外）— 图标使用纯色填充或描边
6. **viewBox 规范** — 所有 SVG 必须设置 viewBox="0 0 48 48" 或对应比例
7. **无 px 单位** — SVG 内部使用纯数字坐标

## 四、背景图设计需求

### 1. `bg-home.svg` — 首页 Hero 背景
**尺寸：** 1440×900（或 16:9 比例，可缩放）
**格式：** SVG
**描述：**
- 整体是无名岛的远景视角，从草地仰望天空
- **天空：** 上半部分天蓝色到浅蓝色的渐变，点缀 3-5 朵圆润的卡通白云
- **中景：** 一层柔和起伏的绿色山丘，山丘上有几棵简化的松树（三角形+矩形组合，但要圆润化）
- **前景：** 一片温暖的绿色草地，草地上有几朵小花（粉色、黄色、白色小圆点）
- **氛围：** 左下角或右上角有一轮温暖的卡通太阳，带粗圆光芒线
- **色调：** 整体偏暖绿 `#7DC395` 到 `#6fb586`，不是冷绿
- **风格：** 扁平插画，线条圆润，无复杂细节，像动森的加载画面背景

### 2. `bg-content.svg` — 内容页背景纹理
**尺寸：** 400×400（无缝平铺）
**格式：** SVG
**描述：**
- 底色：`#f8f8f0`（奶油米白）
- 纹理：极淡的横向细线，像笔记本纸的横线，线色 `#e8e2d6`，opacity 0.3
- 或者：极淡的小圆点矩阵，间距大，点色 `#e8e2d6`，opacity 0.2
- 必须是无缝平铺的（seamless tile）

### 3. `bg-auth.svg` — 登录/注册页背景
**尺寸：** 1440×900
**格式：** SVG
**描述：**
- 室内温馨场景，像动森玩家的小屋
- 左侧：一张木质书桌的一角，桌上有一盏暖黄色台灯（发光效果用浅色椭圆）
- 右侧：一扇窗户，窗外是夜晚的深蓝色天空，有几颗圆润的小星星 `#ffcc00`
- 地板：浅色木地板纹理（用平行粗线表示）
- 整体色调：暖黄 `#f8f8f0` 到 `#f0e8d8`，温馨安全感

## 十、设计参考资料

### 动森风格参考图（文字描述）
- **NookPhone 图标：** 圆角方形（border-radius 45px），图标居中，线条粗圆，颜色鲜艳但柔和
- **村民角色：** 圆润的头部，豆豆眼或简单的圆点眼睛，小圆鼻子，没有尖锐的棱角
- **对话框：** 有机不规则的 blob 形状，边缘有轻微的波浪
- **道具图标：** 极度简化，只保留最核心的特征，用粗圆线条勾勒

### 禁止出现的设计元素
- 直角矩形
- 冷灰色系（#999, #ccc 等冷调灰）
- 细线条（stroke-width < 2px）
- 真实照片/写实风格
- 复杂的渐变（简单的两色渐变可以）
- 阴影效果（SVG 内不使用 drop-shadow，保持扁平）
- 尖锐的三角形（所有角必须圆润）

## 十一、技术规范（设计师必读）

### SVG 导出设置
```
- 预设：SVG 1.1
- 字体：转曲（outline）
- 图像：不嵌入
- 对象 IDs：使用图层名称
- 小数精度：2 位
- 样式：内联样式（style attributes）
- 压缩：使用 svgo 或类似工具优化
```

### 命名图层规范
每个图标文件内部，图层/组名应规范为：
```
icon-{name} /
  ├── outline (描边层)
  ├── fill (填充层)
  └── detail (细节/点缀层)
```

### 尺寸检查清单
- 所有 48×48 图标实际内容区域为 40×40，四周留白 4px
- 所有 64×64 图标实际内容区域为 54×54，四周留白 5px
- 所有 24×24 图标实际内容区域为 20×20，四周留白 2px
- stroke-linecap: round
- stroke-linejoin: round
- 无 px 单位
